# Frappe Fabric

Fabric and Roll Management for ERPNext.

## Installation

```bash
bench get-app frappe_fabric
bench --site [site-name] install-app frappe_fabric
bench --site [site-name] migrate
```

## Features

- Fabric Roll Management with QR code tracking
- Container Receipt processing
- Sample Cutting and Retail Sales
- Roll Transfer between warehouses
- Stock Balance and Movement Reports
- Real-time roll location tracking
- Batch and Color management
